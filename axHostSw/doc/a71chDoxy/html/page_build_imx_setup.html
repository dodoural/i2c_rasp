i.MX6UL build example application
Note
The software is structured in library components, example functions and an application.
The procedures outlined in this section will create one executable out of all these source files.
Refer to i.MX6UL/POSIX Build for more elaborate information on building the A71CH Host Sw (a.o. info on building a separate library with executables that link to it).

The source code is generic and contains C preprocessor statements relying on the definition of the following macro names.

  • -DSCP_MODE=C_MAC_C_ENC_R_MAC_R_ENC
  • -DOPENSSL
  • -DTGT_A71CH
  • -DHOST_CRYPTO
  • -DLINUX
  • -DI2C
  • -DPLATFORM_IMX

Compilation for i.MX6UL: Cross Compile

To build the example application with a Yocto created cross-compiler (located in e.g. /opt/fsl-imx-fb/4.9.11-1.0.0, please note your specific location of the tool may be different): 

cd ~/…/axHostSw/linux
source /opt/fsl-imx-fb/4.9.11-1.0.0/environment-setup-cortexa7hf-neon-poky-linux-gnueabi
make -f Makefile_A71CH default app=A71CH conn=i2c platf=imx

Copy the resulting binary onto the iMX target’s filesystem.
Specific information on preparing the development environment for NXP’s MCIMX6UL-EVKB is provided in Setup i.MX6ULEVK.
Please consult the documentation that comes with the platform (http://www.nxp.com/imx6ULEVK) for a comprehensive and detailed introduction.

Compilation for i.MX6UL: Native Compile

Note
The OpenSSL libraries for the embedded platform are assumed to be available.

To build the example application with a self-hosted gcc-toolset om i.MX6UL, the procedure is similar to the one outlined for cross-compilation. The same makefile (Makefile_A71CH) is used. To build the application on i.MX6UL (on a prepared platform) invoke the makefile as follows: 

cd ~/…/axHostSw/linux
make -f Makefile_A71CH default app=A71CH conn=i2c platf=imx

Specific information on preparing a native development environment for the i.MX6UL is provided in Setup i.MX6ULEVK (please also check the appendix).
Please consult the documentation that comes with the platform (http://www.nxp.com/imx6ULEVK) for a comprehensive and detailed introduction.
