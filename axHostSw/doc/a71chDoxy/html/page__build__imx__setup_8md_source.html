1 # i.MX6UL build example application {#page_build_imx_setup}
2 
3 
4 \note The software is structured in library components, example functions and an application. \n
5 The procedures outlined in this section will create one executable out of all these source files. \n
6 Refer to \ref posix_build for more elaborate information on building the A71CH Host Sw (a.o. info on building a separate library with executables that link to it).
7 
8 The source code is generic and contains C preprocessor statements relying on the definition of the following macro names.
9 
10 - `-DSCP_MODE=C_MAC_C_ENC_R_MAC_R_ENC`
11 - `-DOPENSSL`
12 - `-DTGT_A71CH`
13 - `-DHOST_CRYPTO`
14 - `-DLINUX`
15 - `-DI2C`
16 - `-DPLATFORM_IMX`
17 
18 \section sec_build_imx6ul Compilation for i.MX6UL: Cross Compile
19 
20 To build the example application with a Yocto created cross-compiler (located in e.g. /opt/fsl-imx-fb/4.9.11-1.0.0, please note your specific location of the tool may be different):
21 
22  cd ~/…/axHostSw/linux
23  source /opt/fsl-imx-fb/4.9.11-1.0.0/environment-setup-cortexa7hf-neon-poky-linux-gnueabi
24  make -f Makefile_A71CH default app=A71CH conn=i2c platf=imx
25 
26 Copy the resulting binary onto the iMX target’s filesystem. \n
27 Specific information on preparing the development environment
28 for NXP’s MCIMX6UL-EVKB is provided in \ref page_imx6ul. \n
29 Please consult the documentation that comes with the platform (http://www.nxp.com/imx6ULEVK) for a comprehensive and detailed introduction.

32 \section sec_build_imx6ul_native Compilation for i.MX6UL: Native Compile
33 
34 \note The OpenSSL libraries for the embedded platform are assumed to be available.
35 
36 To build the example application with a self-hosted gcc-toolset om i.MX6UL, the procedure is similar to the one outlined for cross-compilation.
37 The same makefile (Makefile_A71CH) is used.
38 To build the application on i.MX6UL (on a prepared platform) invoke the makefile as follows:
39 
40  cd ~/…/axHostSw/linux
41  make -f Makefile_A71CH default app=A71CH conn=i2c platf=imx
42 
43 Specific information on preparing a native development environment for the i.MX6UL is provided in \ref page_imx6ul (please also check the appendix). \n
44 Please consult the documentation that comes with the platform (http://www.nxp.com/imx6ULEVK) for a comprehensive and detailed introduction.
