MSVC Examples

Microsoft Visual Studio Projects are available for the following combinations.

visual studio version OpenSSL Socket OpenSSL vcom mbedtls vcom
- VS2010 yes yes yes
- VS2012 yes yes yes
- VS2015 yes yes yes
- VS2017 yes yes yes

There are example projects in each solution. Please refer to individual example project documentation for further details about the example.

for e.g. To build the A71CH mbedtls vcom example projects using MS Visual Studio 2010 open the solution file
<install_dir>/projects_vs2010/A71CH_socket_mbedtls/_mbedtls_a71ch_socket_x86.sln
Invoke menu Build > Build Solution to create the executables.
executables will be created for all example projects The resulting executable for e.g. mbedTLS_sa_dtls_server.exe will be located in directory bin/Debug of the corresponding solution.

Communication between PC and A71CH

mbedTLS projects establish a communication channel with the target hardware via the VCOM interface. Please refer to VCOM page for more details

OpenSSL Projects can establish a communication channel with the target hardware using VCOM as well as RJCT socket. For more details about RJCT Socket please refer to RJCT Server page

To run and debug VCOM examples, VCOM firmware has to be running on target hardware

The OpenSSL libraries for the MS Windows platform are pre-built. It is located in folders:

  • <install_dir>/ext/openssl/lib_vs2010 and
  • <install_dir>/ext/openssl/lib_vs2015

Visual Studio 2010 and 2012 projects use the library in lib_vs2010 folder. Visual Studio 2015 and 2017 projects use the library in lib_vs2015 folder.

_openssl_a71ch_vcom_x86 and _openssl_a71ch_socket_x86 are avbailable only for OpenSSL based build configuration.

Project name Description
A71CHConfig This is an example project to configure A71CH.

The A71CH Configure Tool is an application supporting the injection of credentials into the A71CH. It can also report on the value and status of the stored credentials. It also reports the status of the device.
A71CH Configure Tool is installed on a PC, and the communication with the FRDM_K64_F containing the A71CH is achieved over TCP/IP Socket or VCOM.
Please refer to A71CH Configure Tool for additional details about the config tool.
