3 # MSVC Examples {#sec_build_windows}
4 
5 Microsoft Visual Studio Projects are available for the following combinations.
6 
7 visual studio version | OpenSSL Socket | OpenSSL vcom | mbedtls vcom
8 ----------------------|----------------|--------------|-------------
9 - VS2010 | yes | yes | yes
10 - VS2012 | yes | yes | yes
11 - VS2015 | yes | yes | yes
12 - VS2017 | yes | yes | yes
13 
14 There are example projects in each solution. Please refer to individual example project documentation for further details about the example.
15 
16 for e.g. To build the A71CH mbedtls vcom example projects using MS Visual Studio 2010 open the solution file \n
17 \p &lt;install_dir&gt;/projects_vs2010/A71CH_socket_mbedtls/_mbedtls_a71ch_socket_x86.sln \n
18 
19 Invoke menu Build > Build Solution to create the executables. \n
20 executables will be created for all example projects
21 The resulting executable for e.g. \p mbedTLS_sa_dtls_server.exe will be located in directory \p bin/Debug of the corresponding solution.

23 # Communication between PC and A71CH {#hardware_communication}
24 
25 mbedTLS projects establish a communication channel with the target hardware via the VCOM interface.
26 Please refer to \ref page_VCOM page for more details
27 
28 OpenSSL Projects can establish a communication channel with the target hardware using VCOM as well as RJCT socket.
29 For more details about RJCT Socket please refer to \ref page_rjct_server page
30 
31  To run and debug VCOM examples, VCOM firmware has to be running on target hardware
32 
33 
34 The OpenSSL libraries for the MS Windows platform are pre-built.
35 It is located in folders:
36 - \p &lt;install_dir&gt;/ext/openssl/lib_vs2010 and
37 - \p &lt;install_dir&gt;/ext/openssl/lib_vs2015
38 
39 Visual Studio 2010 and 2012 projects use the library in `lib_vs2010` folder.
40 Visual Studio 2015 and 2017 projects use the library in `lib_vs2015` folder.

67 # OpenSSL specific projects/solution {#openSSL_Proj}
68 
69 \b _openssl_a71ch_vcom_x86 and \b _openssl_a71ch_socket_x86 are avbailable only for OpenSSL based
70 build configuration.
71 
72  Project name | Description |
73  ----------------|-------------------------------------------------------------|
74  A71CHConfig | This is an example project to configure A71CH.
75 
76 The A71CH Configure Tool is an application supporting the injection of credentials into
77 the A71CH. It can also report on the value and status of the stored credentials. It also reports the
78 status of the device.\n
79 A71CH Configure Tool is installed on a PC, and the communication with the FRDM_K64_F containing the A71CH is
80 achieved over TCP/IP Socket or VCOM.\n
81 Please refer to \ref page_a71ch_configure_tool for additional details about the config tool.
