◆ HLSE_SendAPDU()

HLSE_RET_CODE HLSE_SendAPDU ( U8 *  cmd,
U16  cmdLen,
U8 *  resp,
U16 *  respLen 
)

Sends the command APDU to the Secure Element and retrieves the response APDU. The latter consists of the concatenation of the response data (possibly none) and the status word (2 bytes).

The command APDU and response APDU are not interpreted by the host library.

The command/response APDU sizes must lay within the APDU size limitations

Parameters
[in]cmdcommand APDU
[in]cmdLenlength (in byte) of cmd
[in,out]respresponse APDU (response data || response status word)
[in,out]respLenIN: Length of resp buffer (resp) provided; OUT: effective length of response retrieved.
Return values
HLSE_SW_OKUpon successful execution
HLSE_ERR_API_ERRORInvalid function arguments
